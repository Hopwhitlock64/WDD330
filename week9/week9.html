    Notes

  • requestAnimationFrame does run the animation in the browser when the tab is not running/ is inactive which helps smooth the performance
  • The frame rate cannot be set using requestAnimationFrame()/li>
  • You can use setInterval to set a time limit when working with animations
  • windows. methods are useful when in a broswer setting but doesn't work well with smaller screens
  • The data- attribute is a way of embedding data in a web page using custom attributes that are ignored by the browser
  • The names of the attributes must use the following format... start with data- , Contain only lowercase letters, numbers, hyphens, dots, colons or underscores, Include an optional string value

    • Example: data-user = 'Awesome' data-max-length = '45' data-rating = '6'

  • Using web storage api to store data is similar to using cookies
  • The information is saved in the form of key-value pairs, and the values can only be strings. The information is saved to localStorage
  • Web workers allow processes to be run in the background, adding support for concurrency in JavaScript
