    Notes

  • You can use fetch() to post json data
  • Files can be uploaded using an HTML input type="file" input element, FormData() and fetch()
  • You can upload multiple files
  • You can enable client and server side validation in forms
  • You can validate forms without using a built-in API
  • important questions to ask yourself when you want to validate a form:

    • What kind of validation should I perform? What should I do if the form doesn't validate? How can I help the user to correct invalid data?

  • Request() accepts exactly the same parameters as the fetch() method
